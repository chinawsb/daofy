# Delphi 知识库集成配置指南

## 📋 概述

Delphi MCP Server 已经集成了完整的知识库功能，提供多种类型的知识库支持：

| 知识库类型 | 说明 | 数据规模 |
|-----------|------|---------|
| **Delphi 官方源码知识库** | Delphi 官方源代码 | 55,835 类 / 193,335 函数 / 2,897 文件 / 151.05 MB |
| **第三方库知识库** | 第三方组件库源码 | 12,381 类 / 56,524 函数 / 4,768 文件 / 42.07 MB |
| **帮助文档知识库** | Delphi CHM 帮助文档 | 169,788 文档 / 18,297 类 / 28,477 函数 / 182,484 代码示例 / 126.85 MB |
| **项目知识库** | 针对特定项目的知识库 | 动态构建 |

## ✅ 安装状态检查

### 1. 依赖检查
- **Python环境**: Python 3.10+ 已安装
- **MCP包**: mcp >= 1.0.0 已安装
- **其他依赖**: beautifulsoup4, html2text, lxml 已安装
- **虚拟环境**: venv 目录已存在并配置完成

### 2. 知识库目录结构
```
data/
├── delphi-knowledge-base/          # Delphi 官方源码知识库
│   └── index/
│       ├── knowledge_base_vector.sqlite
│       ├── metadata.json
│       └── source_index.json
├── thirdparty-knowledge-base/      # 第三方库知识库
│   └── index/
│       ├── knowledge_base_vector.sqlite
│       ├── metadata.json
│       ├── source_index.json
│       └── thirdparty_paths.json
├── help-knowledge-base/            # 帮助文档知识库
│   ├── extracted/                  # 解压后的 HTML 文件
│   └── index/
│       ├── knowledge_base_vector.sqlite
│       └── metadata.json
└── projects/                       # 项目知识库
    └── <project-name>/
        ├── project_kb/             # 项目源码知识库
        └── thirdparty_kb/          # 项目第三方库知识库
```

## 🚀 快速开始

### 1. 配置 AI 助手

#### Trae 配置
**配置文件**: `C:\Users\<用户名>\.trae-cn\mcp_config.json`

```json
{
  "mcpServers": {
    "delphi-compiler": {
      "command": "F:\\ProPlus\\DelphiPlus\\Experts\\DelphiMCPServer\\delphi-complier-mcp-server\\venv\\Scripts\\python.exe",
      "args": [
        "F:\\ProPlus\\DelphiPlus\\Experts\\DelphiMCPServer\\delphi-complier-mcp-server\\src\\server.py"
      ],
      "env": {
        "PYTHONUNBUFFERED": "1",
        "PYTHONIOENCODING": "utf-8",
        "PYTHONUTF8": "1"
      }
    }
  }
}
```

#### Claude Desktop 配置

```json
{
  "mcpServers": {
```

### 2. 构建知识库

#### 构建 Delphi 官方源码知识库
```
请构建 Delphi 官方源码知识库
```
或使用工具：`mcp_delphi-compiler_build_knowledge_base`

#### 构建第三方库知识库
```
请构建第三方库知识库
```
或使用工具：`mcp_delphi-compiler_build_thirdparty_knowledge_base`

#### 构建帮助文档知识库
```
请构建帮助文档知识库
```
或使用工具：`mcp_delphi-compiler_build_help_knowledge_base`

**分步骤构建**（推荐用于大知识库）：
1. `extract_help_chm` - 解压 CHM 文件
2. `scan_help_html` - 扫描 HTML 文件
3. `build_help_kb_index` - 构建向量索引

#### 初始化项目知识库
```
请初始化项目知识库，项目路径: C:\path\to\project.dpr
```
或使用工具：`mcp_delphi-compiler_init_project_knowledge_base`

## 🔧 可用工具

### Delphi 官方源码知识库工具

| 工具名 | 描述 | 参数 |
|-------|------|------|
| `build_knowledge_base` | 构建 Delphi 源码知识库 | `version`, `force_rebuild` |
| `search_class` | 搜索类定义（支持 class/record/interface/enum） | `class_name` |
| `search_function` | 搜索函数/过程定义 | `function_name` |
| `semantic_search` | 语义搜索 | `query`, `top_k` |
| `get_knowledge_base_stats` | 获取统计信息 | - |
| `list_delphi_versions` | 列出已安装的 Delphi 版本 | - |

### 第三方库知识库工具（全局）

| 工具名 | 描述 | 参数 |
|-------|------|------|
| `build_thirdparty_knowledge_base` | 构建第三方库知识库 | `force_rebuild` |
| `search_thirdparty_class` | 搜索类/record/interface/enum | `class_name` |
| `search_thirdparty_function` | 搜索函数 | `function_name` |
| `search_thirdparty_record` | 搜索 record 类型 | `record_name` |
| `semantic_search_thirdparty` | 语义搜索 | `query`, `top_k` |
| `search_by_filename` | 按文件名搜索（支持通配符） | `filename` |
| `get_thirdparty_kb_stats` | 获取统计信息 | - |
| `get_thirdparty_paths_global` | 获取第三方库路径列表 | - |

### 帮助文档知识库工具

| 工具名 | 描述 | 参数 |
|-------|------|------|
| `build_help_knowledge_base` | 完整构建（解压+扫描+索引） | `help_names`, `async_mode`, `max_files_per_help` |
| `extract_help_chm` | 解压 CHM 文件（第1步） | `help_names` |
| `scan_help_html` | 扫描 HTML 文件（第2步） | `help_names`, `max_files_per_help`, `source_dir` |
| `build_help_kb_index` | 构建向量索引（第3步） | `help_names`, `max_files_per_help`, `source_dir`, `async_mode` |
| `get_task_status` | 获取后台任务状态 | `task_id` |
| `list_tasks` | 列出所有后台任务 | - |
| `search_help` | 搜索帮助文档 | `query`, `top_k` |
| `get_help_kb_stats` | 获取统计信息 | - |

### 项目知识库工具

## 🛠️ 故障排除

### 1. 知识库未构建
```
请构建 Delphi 官方源码知识库
请构建第三方库知识库
```

### 2. 强制重建知识库
```
请强制重建 Delphi 官方源码知识库
```

### 3. 帮助文档知识库构建失败
- 确保已安装 7-Zip
- 使用分步骤构建方式
- 检查 Delphi 帮助目录是否存在

### 4. 项目知识库初始化失败
- 确保项目文件 (.dpr/.dproj) 存在
- 检查项目路径是否正确
- 确保有写入权限

### 5. 搜索无结果
- 确认知识库已构建完成
- 尝试使用语义搜索
- 检查搜索关键词是否正确
