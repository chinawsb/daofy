左右道飞MCP服务系统（Daofy MCP Server）是一款基于 Model Context Protocol（MCP）的服务器软件，用于连接 AI 助手与 Delphi 开发环境，实现智能化的项目编译、知识库检索和代码编辑操作。

**项目规模：**
- 源文件：71 个源文件（config/ 1 文件/592 行；models/ 6 文件/477 行；root/ 3 文件/1499 行；services/ 21 文件/15836 行；tools/ 28 文件/15489 行；utils/ 12 文件/3399 行）
- 测试文件：45 个
- 总代码行数：约 37,292 行

**核心功能：**

1. **Delphi项目编译管理**：支持通过 MSBuild 和 dcc32 编译器编译完整的 Delphi 工程，自动从 Windows 注册表检测已安装的 20 个版本的 Delphi 编译器。同时支持单文件编译、编译事件处理和命令行参数预览，为 AI 辅助开发提供可靠的编译能力。

2. **知识库语义检索**：内置 Delphi 官方 RTL/VCL/FMX 源码知识库，支持类名搜索、函数搜索和语义向量搜索。采用增量构建机制自动追踪三方库和项目源码的变更，确保知识库始终与代码保持同步。

3. **Delphi文件操作**：支持 .pas/.dfm/.dproj 等 Delphi 专用文件的读写、格式化和备份管理。内置编码自动检测功能，支持 UTF-8/UTF-16/GBK 等多编码格式，批量写入功能可一次完成多处代码修改。

4. **DFM组件可视化编辑**：提供组件增、删、改、生成功能，修改 DFM 表单文件后自动同步 PAS 文件中的声明代码。支持创建运行时动态组件，并通过编译验证组件代码的正确性。

5. **多平台代码托管**：统一操作 Gitea、GitHub、GitLab、Gitee 和 GitCode 五大代码托管平台的 API 接口。同时支持 Git 本地操作（status/add/commit/push/clone），提供推送失败自动重试机制。

6. **AI经验记忆管理**：支持 AI 助手保存和搜索成功解决问题的做法，采用语义向量嵌入自动去重，相似度大于 0.85 的经验自动合并到旧记录。提供定期清理低价值条目和合并抽象经验的能力，持续沉淀开发知识。

7. **编码规范查询**：提供 Delphi 源码编码规范的分段查询功能，涵盖命名规则、格式化规则、修改规则和审核规则。支持项目级别的自定义编码规则，按优先级覆盖默认规则，为 AI 代码生成提供规范依据。

8. **著作权文档自动生成**：自动生成软件著作权申请所需的源代码文档、用户手册和申请信息汇总表。集成 Mermaid 图表渲染和浏览器 PDF 导出功能，支持草稿审计和配置完整性校验。

**技术特点：**

1. **MCP协议通信**：基于 Model Context Protocol（JSON-RPC 2.0）实现 AI 助手与 Delphi 开发环境的标准化通信，支持工具注册、资源访问和提示模板三大 MCP 核心能力。

2. **多引擎知识库**：采用 SQLite FTS5 全文检索引擎与向量嵌入（embedding）搜索引擎双引擎架构，FTS5 提供精确的关键词匹配，向量引擎支持语义相似度搜索。

3. **Windows多进程管理**：使用 ProcessPoolExecutor 管理编译和知识库构建的并发子进程，严格控制子进程超时（30 秒长轮询）并确保资源清理。Worker 进程内部禁用 stdout 打印，避免污染 MCP 的 JSON-RPC 通信通道。

4. **编译器自动检测**：通过 Windows 注册表自动检测已安装的 Delphi 编译器，支持 Delphi 2005 至最新版共 20 个版本，无需手动配置路径。检测失败时自动回退默认路径列表，确保开箱即用。

5. **增量构建与缓存**：基于文件时间戳和 MD5 哈希的增量更新机制，仅重新处理变更的源码文件。构建结果按文档类型分库存储，支持热切换重建且不阻塞当前查询。

**应用场景：**

1. **AI辅助Delphi开发**：开发者通过自然语言指令即可完成 Delphi 项目编译、知识库搜索和代码编辑操作，无需手动切换 IDE 或命令行。适用于快速原型开发、代码审查和日常编码场景。

2. **代码审查与质量控制**：结合静态分析审计（audit）、代码骨架提取（AST）和运行时检测（runtime check）三项能力，自动检测遗漏的 uses 引用单元、资源泄露风险和代码规范违规。

3. **遗留系统维护**：对使用旧版本 Delphi（2005—2010）开发的项目，提供语义搜索和知识库构建能力，帮助 AI 快速理解大型遗留代码库的结构和依赖关系。支持增量更新，适应持续维护场景。

4. **软件著作权申请辅助**：自动生成软件著作权申请所需的全部文档材料，包括源代码 PDF、说明书 PDF 和申请信息汇总表。内置草稿审计功能，提前发现材料不完整或逻辑不一致的问题，减少驳回风险。
