### 8.5 协同工作流

```
AI Agent                            人工（开发者）
   │                                      │
   │── 独立尝试解决问题 ──────────────→       │
   │── 提交介入申请书（按 8.4 清单）──→       │
   │←── 人工响应（定位根因/提供信息/授权）──  │
   │── 根据反馈继续修复 → 验证 ──────→       │
   │←── 验收确认（通过或调整）───────────     │
```

**关键规则**：
1. **AI 必须主动申请介入** — 满足 8.1 条件立即触发，不浪费时间
2. **介入申请必须完整** — 缺 8.4 任意一项可被退回补充
3. **AI 继续跟进** — 人工提供方案后 AI 负责执行修复并验证
4. **经验沉淀** — 解决后 `experience(action="save")` 保存，下次可自行解决
5. **介入后更新规则** — 规则盲区 → 更新 `src/resources/coding-rules.md`
