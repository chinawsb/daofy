### 8.4 人工介入交接清单

申请介入时，以下信息**缺一不可**：

```
┌────────────────────────────────────────────┐
│               人工介入申请书                  │
├────────────────────────────────────────────┤
│ [一] 问题摘要                               │
│   一句话描述问题                             │
│                                            │
│ [二] 已尝试方案（逐条列出）                    │
│   ① 尝试了什么 → 结果如何 → 为什么失败          │
│   ② ...                                     │
│   ③ ...                                     │
│                                            │
│ [三] 异常现场（见 8.3 模板）                   │
│   ├─ 完整异常消息 + 调用栈                    │
│   ├─ 关键变量值                              │
│   └─ 复现步骤（从启动到异常的最简路径）           │
│                                            │
│ [四] 环境信息                                │
│   ├─ Daofy/Python/OS 版本                   │
│   ├─ Delphi 编译器版本                       │
│   ├─ 项目类型（VCL/FMX/Console）              │
│   └─ 最近变更（与问题相关）                     │
│                                            │
│ [五] 预期结果                                │
│   希望人工具体做什么                           │
│                                            │
│ [六] 最小复现示例（MRE）                       │
│   如能剥离，附上代码                           │
└────────────────────────────────────────────┘
```
